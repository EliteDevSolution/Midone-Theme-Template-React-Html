Style Customization

CSS Structure

Midone use PostCSS as a default CSS transformation tool, you can found all of the CSS files in one single place at /src/assets/css/* directory. Every single component CSS file are imported in /src/assets/css/app.css. You can modify this file to add or remove components depends to your needs. Here is the part things you need to know about the /src/assets/css/app.css structure and functions.

  • Colors: This file is contains configuration of the main and dark-mode colors.
  • Fonts: This section is supposed to import all the template used fonts.
  • 3rd Parties: This section supposed to import all 3rd parties CSS or CSS files.
  • Components: This section supposed to import all component CSS files used by the template. 
            
                /*
                |--------------------------------------------------------------------------
                | TailwindCSS Directives
                |--------------------------------------------------------------------------
                |
                | Import TailwindCSS directives and swipe out at build-time with all of
                | the styles it generates based on your configured design system.
                |
                | Please check this link for more details:
                | https://tailwindcss.com/docs/installation#include-tailwind-in-your-css
                |
                */

              /*
                |--------------------------------------------------------------------------
                | Colors
                |--------------------------------------------------------------------------
                |
                | Configuration of the main and dark-mode colors.
                |
                | Please check this link for more details:
                | https://tailwindcss.com/docs/customizing-colors#using-css-variables
